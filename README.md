## Installation

Install package laravel & tailwindcss

```sh
# composer install
# npm install
```

Set web application & database

```sh
# cp .env.example .env
# php artisan key:generate
```

Migrate and seed database

```sh
# php artisan migrate
# php artisan db:seed
```
For running

```sh
# php artisan serv
# npm run dev
```
